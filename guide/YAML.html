# YAML

YAML (YMAL Ain't Markup Language)

​ YAML A Markup Language: 是一个标记语言

​ YAML isn'not Markup Language: 不是一个标记语言

标记语言：

​ 以前的配置文件；大多都使用的是xxxx.xml文件；

​ YAML以数据为中心，比json，xml更适合做配置文件；

YAML：配置服务器端口实例

server:
	port: 8081

XML：配置服务器端口

<server>
	<port>8081</port>
</server>

# YMAL语法:

# 1.基本语法

key:(空格)value :表示一堆键值对（空格必须有）

以空格的缩进来控制层级关系；只要是左对齐的一列数据都是同一个层级的

server:
    prot: 8081
    path: /hello

属性和值也是大小写敏感；

# 2.值的写法

# 字面量：普通的值（数字，字符串，布尔）

​ k: v :字面直接来写；

​ 字符串默认不用加上单引号或双引号；

​ "":双引号不会转义字符串中的特殊字符，会输出特殊字符本身想表达的意思；

name: "zhangsan \n lisi";双引号包裹下会将\n转换成换行，会输出: zhangsan 换行 list

​ '':单引号会转义字符串中的特殊字符.特殊字符最终只是一个普通的字符串；

name: 'zhangsan \n lisi';单引号包裹下不会将\n转换成换行，会输出: zhangsan \n list

# 对象，Map（属性和值）（键值对）

​ k: v:在下一行写对象的属性和值的关系，注意缩进

​ 对象也是k: v的方式

friends: 
        name: zhangsan
        age: 20

​ 行内写法：

friends: {name: zhangsan,age: 18}

# 数组（list，set）

​ 用- 值的方式来表示数组中的一个元素

pets:
 - cat
 - dog
 - pig

​ 行内写法

pets: [cat,dog,pig]
