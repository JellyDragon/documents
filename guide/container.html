# 容器

# 1.ArrayList

# 1.1ArrayList常用方法 add方法有两种用法

# 1.1.1 add

1：直接在容器内加入对象

ArrayList.add(object);

2：在容器指定位置加入对象

ArrayList.add(index,object);

index为容器下标

# 1.1.2 contains

​ contains能判断一个对象是否在容器内

​ 判断的机制是根据是否是同一个对象，而不是根据name是否一致

ArrayList.contains(objectname);

# 1.1.3 get

​ get可以获得容器内指定位置的对象

ArrayList.get(index);

如果index超过容器下标范围会报错

# 1.1.4 indexOf

​ indexOf可以过的容器内指定对象的位置

ArrayList.indexOf(hero);

若对象在容器内不存在会返回-1

# 1.1.5 remove

​ remove可以删除容器内对象

​ 可以通过对象引用删除

ArrayList heros = new ArrayList();
Hero spe = new Hero("special hero");
        heros.add(3,spe);
heros.remove(spe);

​ 可以通过容器下标删除

ArrayList heros = new ArrayList();
heros.remove(1);

# 1.1.6 size

​ size可以获取容器的大小

ArrayList.Size();

# 1.1.7 toArray

​ toArray可以将容器转化为数组

Hero[] hs = (Hero[]) heros.toArray(new  Hero[]{});

# 1.1.8 addAll

​ addAll方法可以将一个容器中的所有元素插入到另一个容器中

ArrayList heros = new ArrayList();

        for (int i=0;i<5;i++){
            //add方法直接增加对象
            heros.add(new Hero("hero"+i));
        }
        System.out.println(heros);
ArrayList hs = new ArrayList();
        for (int i=0;i<5;i++){
            //add方法直接增加对象
            heros.add(new Hero("hero"+i+10));
        }
        heros.addAll(hs);
        System.out.println(heros);

# 1.1.9 clear

​ clear方法可以将一个容器中的元素清空

ArrayList.clear();

# 2 LinkedList

​ LinkedList也实现了List接口，ArrayList的常用方法同样支持

# 2.1 Deque

​ 除了实现了List接口外，LinkedList还实现了双向链表结构Deque，可以很方便的在头尾插入删除数据。

​ LinkedList因为是一个双向链表，所以可以对首尾的数据进行操作

# 2.1.1 addFirst与addLast

LinkedList.addFirset(object);

能在容器的头部插入对象

LinkedList.addLast(object);

能在容器的尾部插入对象

# 2.1.2 getFirst与getLast

LinkedList.getFirset();

能查看容器头部的对象

LinkedList.getLast();

能查看容器尾部的对象

# 2.1.3 removeFirst与removeLast

LinkedList.removeFirst()

取出容器头部的对象

LinkedList.removeLast()

取出容器尾部的对象

容器中的对象被取出后会被删除

# 2.2 ArrayList与LinkedList的区别

​ ArrayList是顺序结构 插入，删除数据慢(在数组中插入删除数据要改变插入删除位置之后的所以数据的顺序，执行的操作多)。定位快(就像电影院的位置，每个位置都有相应的座位号)。

​ LinkedList是链表结构，插入，删除数据快(在链表中插入删除数据只需要改变前后两个数据的位置，执行的操作少)。定位慢(就像一串佛珠一样每个都互相连接，想找到数据必须一个一个的数过去)。

​ 比较插入

public static void main(String[] args) {
        List<Integer> arrayList = new ArrayList<>();
        insertFirst(arrayList,"ArrayList");
        List<Integer> linkedList = new LinkedList<>();
        insertFirst(linkedList,"LinkedList");
    }

    public static void insertFirst(List<Integer> i,String type){
        int total = 1000*100;
        final int number = 5;
        long start = System.currentTimeMillis();
        for (int j=0;j<total;j++){
            i.add(0,number);
        }
        long end = System.currentTimeMillis();
        System.out.printf("在%s 最前面插入%d条数据，总共耗时 %d 毫秒 %n", type, total, end - start);
    }

比较定位:

public static void main(String[] args) {
        List<Integer> arrayList = new ArrayList<>();
//        insertFirst(arrayList,"ArrayList");
          modify(arrayList,"ArrayLsit");
        List<Integer> linkedList = new LinkedList<>();
//        insertFirst(linkedList,"LinkedList");
          modify(linkedList,"LinkedLsit");
    }
public static void modify(List<Integer> i,String type){
        int total = 1000*100;
        int index = total/2;
        final  int number = 5;
        //初始化容器
        for (int j =0;j<total;j++){
            i.add(number);
        }
        long start =  System.currentTimeMillis();
        for (int j=0;j<total;j++){
            int n = i.get(index);
            n++;
            i.set(index,n);
        }
        long end = System.currentTimeMillis();
        System.out.printf("%s总长度是%d，定位到第%d个数据，取出来，加1，再放回去%n 重复%d遍，总共耗时 %d 毫秒 %n",
                type,total, index,total, end - start);
    }

# 3 HashMap

​ HashMap通过键值对的方式存储数据

HashMap<String String> hashMap = new HashMap<>();
hashMap.put("key","value");

对于HashMap而言key是唯一的，不可重复的。如果以相同的key把不同的value插入到HashMap中会把之前的数据覆盖，只留下最后插入的数据。HashMap还可以通过不同的key将相同的对象插入到HashMap中。
